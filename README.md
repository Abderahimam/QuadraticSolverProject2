Overview
This project implements a quadratic equation solver in C# and includes unit tests using XUnit.
The solver calculates the roots of a quadratic equation:
ax2+bx+c=0ax^2 + bx + c = 0ax2+bx+c=0
Features:

Handles:

No real roots (discriminant < 0)
One real root (discriminant = 0)
Two real roots (discriminant > 0)


Returns roots sorted in ascending order.
Throws an exception if a = 0 (not a quadratic equation).


 Technologies

.NET 8
C#
XUnit for unit testing
Git & GitHub for version control
